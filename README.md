# Консольное приложение для управления базой данных

**Описание**  
Проект реализует консольную базу данных: создание/удаление таблиц, вставка/чтение/обновление/удаление записей, с
хранением метаданных и данных в JSON-файлах. Реализовано полноценное CRUD-API через командную строку, с валидацией
типов, логированием, кешированием и подтверждением опасных операций.

## Установка

```bash
git clone https://github.com/sargnar/project2_sargsyan_narek_m25_555.git
cd project2_sargsyan_narek_m25_555
make install
# или
poetry install

# Запустить приложение
poetry run project
# или
make run

create_table <имя_таблицы> <столбец1:тип> <столбец2:тип> …      — создать таблицу  
drop_table <имя_таблицы>                                        — удалить таблицу  
list_tables                                                     — показать список всех таблиц  
insert into <имя_таблицы> values (<значение1>, <значение2>, …)  — вставить запись  
select from <имя_таблицы> [where <столбец>=<значение>]          — прочитать записи  
update <имя_таблицы> set <столбец1>=<новое_значение1> where <столбец>=<значение> — обновить  
delete from <имя_таблицы> where <столбец>=<значение>           — удалить запись  
info <имя_таблицы>                                             — информация о таблице  
help                                                           — справка  
exit                                                           — выйти
